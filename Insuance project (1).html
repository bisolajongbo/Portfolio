Introduction¶

Introduction My name is Jongbo Bisola Tolutope and in this project,i will be analyzing insurance claims and my findings from this analysis will be used to build,train and test predict model based on the insurance data

About The Project¶

I have been appointed as the Lead Data Analyst to build a predictive model to determine if a building will have an insurance claim during a certain period or not. I will have to predict the probability of having at least one claim over the insured period of the building. The model will be based on the building characteristics. The target variable, Claim, is a:

1 if the building has at least a claim over the insured period. 0 if the building doesnt have a claim over the insured period.

End-to-End Machine Learning Workflow¶

I will begin with data cleaning by identifying and addressing issues such as inconsistent naming conventions, missing values, incorrect data types, duplicates, and outliers to the best of my ability. Next, I will perform exploratory data analysis to understand how different features influence the target variable and to identify the most relevant features for the model. Finally, I will train and evaluate the model to ensure it performs as expected.

#Feature importance is next
#Check the numerical features, while ignoring the "claim" column to see if they actually have correlation with the target variable

cols = data.select_dtypes(include = np.number).columns.to_list()
#Excluding claim 
cols = [col for col in numerical_cols if col != "Claim" ]

#Calculatete correlations
correlations = []
for col in cols:
    if col in data.columns:
        real = data[[col, "Claim"]].dropna()
        if len(real) > 10:  # Need enough data
            corr = real.corr().iloc[0, 1]
            correlations.append({
                'Feature': col,
                'Correlation': corr,
                'Abs_Correlation': abs(corr),
                'Samples': len(real)
            })

corr_df = pd.DataFrame(correlations).sort_values('Abs_Correlation', ascending=False)

print("Top 10 Features by Absolute Correlation with Claim:")
for i, row in corr_df.head(10).iterrows():
    direction = "positive" if row['Correlation'] > 0 else "negative"
    strength = "strong" if abs(row['Correlation']) > 0.3 else "moderate" if abs(row['Correlation']) > 0.1 else "weak"
    print(f"{row['Feature']:25} {row['Correlation']:7.3f} ({direction}, {strength}) - {row['Samples']} samples")
