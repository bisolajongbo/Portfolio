Project Introduction¶

This project is a modelling project for Yakub Training Group. We are expected to build a model that can predict whether or not someone deserves promotion.

Project Overview¶

As per standard practice, we will clean the data and visualize the data to get an understanding of the data to understand the factors that limit promotion and those that favor promotion, after which we will build base model and add features as necessary.

This project is a solution to one of two assignments given by Mr Israel Odeajo.
